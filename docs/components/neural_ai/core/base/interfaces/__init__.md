Base komponens interfészek.

Ez a modul tartalmazza a Neural AI Next base komponens rendszerének összes interfészét és absztrakt osztályát.

- [`CoreComponentsInterface`](component_interface.md#corecomponentsinterface): Core komponensek interfész
- [`CoreComponentFactoryInterface`](component_interface.md#corecomponentfactoryinterface): Core komponens factory interfész
